Arquitectura del Portafolio

Este sitio web fue diseñado utilizando servicios administrados de AWS, priorizando simplicidad, seguridad y bajo costo.

Servicios utilizados

  • Amazon S3: Almacenamiento del sitio web estático y backend remoto del estado de Terraform.
  • Amazon CloudFront: Distribución global del contenido (CDN) con HTTPS forzado.
  • AWS IAM: Control de acceso entre servicios con el principio de mínimo privilegio.
  • Amazon API Gateway: Punto de entrada HTTPS público para la función de contacto (formulario web).
  • AWS Lambda: Función serverless (Node.js) que procesa los mensajes del formulario y los envía a SNS.
  • Amazon SNS: Servicio de notificaciones que entrega los mensajes del formulario directamente al correo del propietario.
  • Amazon DynamoDB: Tabla de State Locking para prevenir corrupción del estado de Terraform.

Flujo de la arquitectura

El usuario accede al sitio a través de CloudFront usando HTTPS. CloudFront obtiene el contenido estático desde un bucket S3 privado mediante Origin Access Control.

Cuando el usuario envía el formulario de contacto, el navegador hace un POST a API Gateway, que invoca una función Lambda en Node.js. La Lambda publica el mensaje en un Topic de SNS, que lo entrega al correo del propietario en tiempo real.

Decisiones técnicas

  • La infraestructura se gestiona como código (IaC) utilizando Terraform, garantizando despliegues consistentes y reproducibles.
  • Remote State y State Locking: El estado de Terraform se almacena de forma segura en un bucket de Amazon S3 con versionamiento, y las operaciones concurrentes se bloquean utilizando una tabla de Amazon DynamoDB para evitar corrupción del estado.
  • Backend Serverless (Contact API): El formulario de contacto utiliza API Gateway + Lambda + SNS sin servidores que administrar. Costo: $0 dentro del free tier.
  • No se utilizan servidores EC2 para reducir costos y mantenimiento.
  • El bucket S3 principal no es público, siguiendo el principio de mínimo privilegio (restringido vía OAC).
  • CloudFront mejora seguridad, performance y disponibilidad global.
