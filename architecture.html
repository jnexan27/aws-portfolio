Arquitectura del Portafolio

Este sitio web fue diseñado utilizando servicios administrados de AWS, priorizando simplicidad, seguridad y bajo costo.

Servicios utilizados

  • Amazon S3: almacenamiento del sitio web estático
  • Amazon CloudFront: distribución global del contenido (CDN)
  • AWS IAM: control de acceso entre servicios

Flujo de la arquitectura

El usuario accede al sitio a través de CloudFront usando HTTPS. CloudFront obtiene el contenido desde un bucket S3 privado, al cual solo él tiene acceso mediante Origin Access Control.

Decisiones técnicas

  • No se utilizan servidores EC2 para reducir costos y mantenimiento
  • El bucket S3 no es público, siguiendo el principio de mínimo privilegio
  • CloudFront mejora seguridad, performance y disponibilidad
